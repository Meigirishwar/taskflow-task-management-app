# TaskFlow - Task Management Application

## Overview
TaskFlow is a web-based task management application that helps users organize and track daily tasks efficiently.

## Features
- User Registration
- User Login
- Task Creation
- Task Editing
- Task Deletion
- Task Completion Tracking
- Task Priority Management
- Due Date Management
- Local Storage Persistence
- Responsive User Interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage
